Why Cursor?

Understand what Cursor is and why we are using it to build a live midterm countdown page.

Responsible AI National Politics Chat

What is Cursor?

Cursor is a code editor built on VS Code with AI built in. You describe what you want in plain English; Cursor suggests code, edits files, and helps you debug — but you decide what to keep, change, or reject.

You describe a change → Cursor edits files → You review → Push to GitHub Pages → Live on the web

Why “Hello World”?

“Hello World” is the traditional first program — the smallest thing that proves your setup works. Our version goes one step further: a real web page anyone can open in a browser, showing days until the November 3, 2026 midterm elections.

It is deliberately simple: static HTML, CSS, and a few lines of JavaScript. No server, no database, no cost. Hosted free on GitHub Pages.
