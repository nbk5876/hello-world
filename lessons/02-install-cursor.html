Lesson 02

Download & install Cursor

Get Cursor on your computer from the official website. Already installed? Skip to Lesson 03.

Responsible AI National Politics Chat

Skip this lesson if Cursor is already on your machine and opens normally.

Download from the official site only

Go to cursor.com in your browser. Use the Download button on that site — do not install from random links in emails, social posts, or search ads.

Screenshot: cursor.com home page with Download button

Install on Windows

  1. Run the installer file you downloaded (for example CursorSetup.exe).
  2. If Windows SmartScreen asks, confirm you trust the publisher — you downloaded from cursor.com.
  3. Follow the installer prompts (defaults are fine for most people).
  4. When finished, open Cursor from the Start menu or desktop shortcut.

Install on Mac

  1. Open the .dmg file you downloaded.
  2. Drag Cursor into your Applications folder.
  3. Open Cursor from Applications. If macOS asks whether to open it, choose Open.

Sign in (optional)

Cursor may ask you to sign in when you first open it. Signing in enables AI features and syncs some settings. You can explore the editor first and sign in later if you prefer.

You are ready when…

  • Cursor opens without errors
  • You see the editor window — menu bar at the top, sidebar on the left, main area in the center
  • You can type in the chat panel (we will use that in the next lessons)
