Add CSS

Style your page — colors, fonts, and layout — by linking css/style.css from index.html.

How this series works: Cursor creates css/style.css and adds the link in index.html. You run the prompt and review. The sections below help you understand what Cursor built.

What CSS does

CSS is the look of your page: colors, fonts, spacing, and layout. HTML (Lesson 06) holds the words; CSS makes them readable and pleasant on phone and desktop. JavaScript (Lesson 08) will still handle the live day count — CSS does not change that logic.

Remember the three-piece diagram from Lesson 06: index.html stays at the center. Today we add css/style.css and link it from your HTML.

How HTML connects to CSS

Your HTML file must include one line in the <head> that tells the browser to load your stylesheet. It looks like this: 

<link rel="stylesheet" href="css/style.css">

You do not need to type this yourself. In the next section we ask Cursor to add it to index.html when it creates css/style.css. Your job is to check that the line appears in the right place and that href="css/style.css" matches your folder name from Lesson 05 — lowercase css, file named style.css inside it.

Ask Cursor to add the styling

For this step, we ask Cursor to create the CSS file and update index.html with the link line above. You review before you accept.

  1. Open your project and index.html from Lesson 06.
  2. Open Chat.
  3. Paste or adapt this prompt:
Create css/style.css with clean, readable styling for my midterm countdown page: centered layout, a pleasant background color, a large bold style for #days-remaining, and a readable font. Add <link rel="stylesheet" href="css/style.css"> inside the head of index.html. Do not add JavaScript yet.
  1. Confirm Cursor creates only css/style.css and a small edit to index.html — not extra files.
  2. Accept when the link path is exactly css/style.css.
  3. Refresh your page in the browser — it should look styled, not plain black text on white.

Understand the CSS Cursor created

After you accept Cursor’s changes, your stylesheet might look like this — for learning, not for retyping:

body {
  font-family: Arial, Helvetica, sans-serif;
  background: #f4f6f9;
  color: #111827;
  text-align: center;
  padding: 2rem;
}

h1 {
  color: #1a56a0;
}

#days-remaining {
  font-size: 4rem;
  font-weight: bold;
}

Selectors worth knowing

  • body — styles the whole page
  • h1 — styles your main heading
  • #days-remaining — styles the paragraph with that id (the big number later)
Good practice: If styling does not appear, check the link in index.html — wrong folder name or a typo in href is the most common mistake. Open DevTools (F12) only if you are curious; fixing the path usually solves it.
