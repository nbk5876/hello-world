What JavaScript does here

JavaScript is the behavior of your page. HTML (Lesson 06) holds the structure; CSS (Lesson 07) handles the look. JavaScript calculates how many days remain until Tuesday, November 3, 2026 and writes that number into the paragraph with id="days-remaining" — the placeholder dash from Lesson 06.

See the working example on the home page: nbk5876.github.io/hello-world/

How HTML connects to JavaScript

Your HTML file must include one line near the end of the body that tells the browser to load your script. It looks like this: 

<script src="js/countdown.js"></script>

You do not need to type this yourself. In the next section we ask Cursor to add it to index.html when it creates js/countdown.js. Your job is to check that the line appears just before </body> and that src="js/countdown.js" matches your folder name from Lesson 05 — lowercase js, file named countdown.js inside it.

Understand the JavaScript Cursor created

After you accept Cursor’s changes, your script might look like this — for learning, not for retyping:

(function () {
  var electionDay = new Date("2026-11-03T00:00:00");
  var today = new Date();
  today.setHours(0, 0, 0, 0);

  var msPerDay = 1000 * 60 * 60 * 24;
  var daysLeft = Math.ceil((electionDay - today) / msPerDay);

  var el = document.getElementById("days-remaining");
  if (!el) return;

  if (daysLeft > 0) {
    el.textContent = daysLeft;
  } else if (daysLeft === 0) {
    el.textContent = "Today";
  } else {
    el.textContent = "Past";
  }
})();

Lines worth knowing

  • new Date("2026-11-03T00:00:00") — the election date JavaScript uses for math
  • document.getElementById("days-remaining") — finds the paragraph you named in Lesson 06
  • el.textContent = daysLeft — replaces the dash with the calculated number
  • Math.ceil(...) — rounds up so partial days count as a full day left
  • (function () { ... })(); — runs the code once when the page loads; keeps variables private
Good practice: If the number does not appear, check two things: the id="days-remaining" in HTML must match exactly what the script looks for, and the <script src="js/countdown.js"> path must point to the right file. Refresh the browser after you accept changes — JavaScript does not update until the page reloads.
